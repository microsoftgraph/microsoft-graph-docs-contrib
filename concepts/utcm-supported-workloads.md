# Supported Resource Types, Properties, Permissions, and Examples
 
Unified Tenant Configuration Management [UTCM] APIs let administrators control and manage configuration settings for the entire organization. The settings can be on one or a combination of settings across the following workloads:
 
- MS Exchange Online
- MS Entra
- MS Teams
- MS Purview
- MS Defender
- MS OneDrive SharePoint
 
UTCM supports 300+ resource types and various properties per resource type. The entire repository of is available in the JSON schema store.
 
For the schema, permissions, and sample examples per resource type, click [here](utcm-supported-resourcetypes.md).
