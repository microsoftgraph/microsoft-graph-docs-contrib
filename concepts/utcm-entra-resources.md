# Supported Microsoft Entra resources for Unified Tenant Configuration Management

This article lists the supported resource types for Microsoft Entra in the unified tenant configuration management (UTCM) APIs in Microsoft Graph. Use these resource types to monitor and manage your Microsoft Entra configuration settings.

For the complete schema, required permissions, and examples for each resource type, see the [UTCM schema store](https://json.schemastore.org/utcm-monitor.json).
