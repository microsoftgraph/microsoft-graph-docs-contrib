> [!IMPORTANT]
> In delegated scenarios with work or school accounts where the signed-in user is acting on another user, they must be assigned one of the following supported [Microsoft Entra roles](/entra/identity/role-based-access-control/permissions-reference?toc=%2Fgraph%2Ftoc.json):
> - Global Secure Access Administrator - read standard properties
> - Security Reader - read standard properties
> - Security Administrator - read standard properties
> - Global Reader
> - Conditional Access Administrator
