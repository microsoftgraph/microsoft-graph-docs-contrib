# webauthnAuthenticationExtensionsClientOutputs resource type

Namespace: microsoft.graph

Contains the output values returned by the client for requested WebAuthn extensions. This is an open type that allows for additional properties based on the specific extensions used. This complex type is the type for the **clientExtensionResults** property of the [webauthnPublicKeyCredential](../resources/webauthnpublickeycredential.md) resource.

This resource is an open type.

## Properties
None.

## Relationships
None.
