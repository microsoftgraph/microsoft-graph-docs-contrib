# cloudPcRegionGroup enum type

Namespace: microsoft.graph

Represents the logical geographic group that a region belongs to. Multiple regions can belong to one region group.

This is an evolvable enumeration. Use the `Prefer: include-unknown-enum-members` request header to get the following values in this [evolvable enum](/graph/best-practices-concept#handling-future-members-in-evolvable-enumerations): `norway`, `switzerland`, `southKorea`.
