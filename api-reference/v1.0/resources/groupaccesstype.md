# groupAccessType enum type

Namespace: microsoft.graph

Indicates the type of access assigned to a group. Used by the **accessType** property of the [group](../resources/group.md) resource.

## Members

| Member | Description |
|:---|:---|
| none | No access type assigned. |
| private | The access type for private groups. |
| secret | The access type for secret groups. |
| public | The access type for public groups. |
| unknownFutureValue | Evolvable enumeration sentinel value. Don't use. |
