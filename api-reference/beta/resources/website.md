# webSite resource type

Represents a website.


### Properties
| Property	   | Type	|Description|
|:---------------|:--------|:----------|
|type|String| Possible values are: `unknown`, `personalHomePage`, `businessHomePage`.|
|url|string|The URL of the website.|
