# markdownBlockMessageConfiguration resource type

Namespace: microsoft.graph.networkaccess

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Markdown body for block message in Global Secure Access policies.


Inherits from [microsoft.graph.networkaccess.blockPageConfigurationBase](../resources/networkaccess-blockpageconfigurationbase.md).


## Properties
|Property|Type|Description|
|:---|:---|:---|
|body|String|Body field of HTML error generated by the Global Secure Access service. Can be programmed in limited Markdown language, with a limit of 1024 characters.|

## Relationships
None.
