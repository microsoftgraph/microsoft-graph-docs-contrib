# complianceDlpSharePointAuditRecord resource type

Namespace: microsoft.graph.security

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Represents an audit record for Compliance Data Loss Prevention (DLP) SharePoint events. This resource captures information about DLP policy matches and actions applied to SharePoint content.


Inherits from [microsoft.graph.security.auditData](../resources/security-auditdata.md).


## Properties
None.

## Relationships
None.
