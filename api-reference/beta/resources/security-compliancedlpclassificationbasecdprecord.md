# complianceDlpClassificationBaseCdpRecord resource type

Namespace: microsoft.graph.security

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

Represents a base audit record for Microsoft Purview Data Loss Prevention (DLP) classification events in the common data platform. This resource provides common properties for common data platform-specific data classification audit records.


Inherits from [microsoft.graph.security.auditData](../resources/security-auditdata.md). The audit data for this record type is returned as the **auditData** property in an [auditLogRecord](../resources/security-auditlogrecord.md).


## Properties
None.

## Relationships
None.
