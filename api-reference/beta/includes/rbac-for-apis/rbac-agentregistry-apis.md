> [!IMPORTANT]
> When using delegated permissions, the authenticated user must be assigned a supported [Microsoft Entra role](/entra/identity/role-based-access-control/permissions-reference?toc=%2Fgraph%2Ftoc.json) or a custom role with a supported role permission. *Agent Registry Administrator* is the least privileged role supported for this operation.
