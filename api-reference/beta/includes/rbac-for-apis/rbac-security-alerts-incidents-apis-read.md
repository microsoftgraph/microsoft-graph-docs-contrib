> [!IMPORTANT]
> In delegated scenarios with work or school accounts, the signed-in user must be assigned a supported [Microsoft Entra role](/entra/identity/role-based-access-control/permissions-reference?toc=%2Fgraph%2Ftoc.json) or a custom role with a supported role permission. The following least privileged roles are supported for this operation.
> 
> - **Security Reader**. Has global read-only access on security-related features, including all information in the Microsoft 365 Defender portal. **This is the least privileged role for this operation.**
> - **Global Reader**. Can read settings and administrative information across Microsoft 365 services, but can't take management actions.
> - **Security Operator**. Can manage alerts and has global read-only access on security-related features, including all information in the Microsoft 365 Defender portal.
> - **Security Administrator**. Has permissions to manage security-related features in the Microsoft 365 Defender portal, including monitoring security-related policies and managing security threats and alerts.
