| ------------ | ----------------------------- | ------------------------ |
| POST, PUT, DELETE, PATCH | 400 requests per 20 seconds | 200 requests per 20 seconds |
| Any | 4000 requests per 20 seconds | 2000 requests per 20 seconds |
